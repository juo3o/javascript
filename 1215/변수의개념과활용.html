변수의 개념과 활용

변수(variable) : 변하는 데이터를 저장하기 위한 일종의 메모리 공간이다. - let / const

변수는 한번에 하나의 데이터만 담을 수 있다.

변수에 대입된 값은 항상 마지막에 저장된 값이 출력된다.

1. 변수의 종류

1) 전역변수 : 문서 전체에 영향을 주는 변수

2) 지역변수 : 함수(function) 안에서만 선언되어 사용되는 변수. 함수안에서만 영향력을 가짐

        let a = 10;              : 전역변수

        function 함수명(){        : 지역변수 ( 팀 작업 시 많이 사용 )
            let b = 5;
        }

2. 변수 선언방법

var , let, const 를 변수명 앞에 붙여준다.

자바스크립트 최신버전 ES6에서는 let, const 사용을 추천한다.

var 변수명 ; 또는 let 변수명; 또는 const 변수명;

var a=10; let b=5; const c=1000;

3. 변수명 작성 시 주의사항

  • 변수명은 영문(대소문자)자로 작성. ex) var a;
  • 변수의 첫글자는 숫자, 기호 사용 안됨. ex) 01a
  • 변수명 사이에는 공백 사용 금지. ex count 01
  • 변수명은 색상이 변하는 단어(변수예약단어)는 사용 금지. var if, break 등등..
  • 변수명ㅇ에는 특수기호 사용금지. (단, $나 _는 사용가능) ex) $Adfdf , crfde_dfdf (스네이크표기법)
  • 변수명은 의미를 담을 수 있는 이름으로 지정, 낙타봉(카멜표기법)으로 작성 권장한다. ex) imageName
