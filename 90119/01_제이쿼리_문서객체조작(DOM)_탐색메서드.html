제이쿼리학습_문서객체조작(DOM)_01객체 탐색(traversing) p.190

제이쿼리에서는 DOM(문서객체모델)구조에서 원하는 요소를 찾고자 할 때 탐색메소드를 사용할 수 있다.

탐색메소드는 기존의 CSS, JQUERY선택자보다 좀더 HTML문서의 요소들을 정확하게 선택할 수 있는 메소드이다.

1. 문서안에서 요소들 중 부모요소 찾기 p.171

  • parent(); - 선택한 요소의 바로 위에 위치한 부모요소를 선택
  • parents(); - 선택한 요소의 바로 위에 있는 모든 부모요소를 선택
  • parentsUntil(); - 선택한 요소에서부터 지정한 요소까지 부모요소를 선택

2. 문서안에서 요소들 중 자식요소 찾기

  • children(); - 선택한 부모요소의 아래 자식요소를 선택.
  • find(); - 선택한 부모요소의 아래에서 해당하는(찾고자하는) 자식요소를 선택

3. 문서안에서 요소들 중 형제요소(siblings) 찾기

  • siblings() - 현재요소의 형제관계에 있는 요소를 모두 선택
  • next() - 현재요소의 다음 요소를 선택
  • nextUntil() - 현재요소에서 다음에서 ~전까지를 선택
  • nextAll() - 현재요소의 다음에 있는 모든 요소를 선택
  • prev() - 현재요소의 이전에 있는 요소를 선택
  • prevAll() - 현재요소의 이전에 있는 모든 요소를 선택
  • prevUntil(); - 현재요소의 이전요소 부터 ~ 전까지를 선택
