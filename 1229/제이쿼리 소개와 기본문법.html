제이쿼리 소개와 기본문법

제이쿼리:

자바스크립트의 생산성을 향상시켜주는 자바스크립트의 '라이브러리' 언어.

자바스크립트 라이브러리 언어 :

자주 사용하는 코드들을 재 사용할 수 있는 형태로 가공하여 프로그래밍의 효율성을 높여주는 코드들의 집합

1. 제이쿼리의 특징

  • 앨리먼트(html태그요소)를 선택하는 강력한 기능을 제공 ( css선택자보다 많음 )
  • 선택된 앨리먼트를 제어할 수 있는 다양한 함수를 제공
  • 자바스크립트의 자주 사용하는 기능을 라이브러리화한 언어

2. 제이쿼리 라이브러리 연결방법

제이쿼리 라이브러리 파일을 직접 연결하기
- 제이쿼리 공식 사이트에서 라이브러리 파일(.js)을 다운로드 받기
- html 문서의 head요소안에 직접 script 작성하여 연결

제이쿼리 라이브러리 CDN방식으로 연결하기
- jquery.com페이지에서 오른쪽 '다운로드'클릭
- 페이지로 이동하고 아래 페이지로 내려가면 'google CDN' 메뉴 클릭

3. 제이쿼리 문법 (작성법)

  • $('선택대상').메서드();
  • jquery('선택대상').메서드();

4. 제이쿼리 라이브러리에서의 $의 역할

제이쿼리에서는 '$'기호를 맨 앞에 넣고 시작. 이것은 괄호()안에 넣을 대상을 선택하기 위함.

ex)

$('h1'), $(window), $('#id'), $("input[type='text']").... window는 객체라서 따옴표없어도됨

5. 제이쿼리에서 '$'기호 안에 들어가는 대상 3가지

  • html태그
  • css선택자
  • 자바스크립트객체 (document, window....)

6. 제이쿼리에서 사용하는 wrapper

  • $(document).ready(function(){실행할내용;});
  • $(function(){실행할내용;});li
  • 밑은 옛날방식...
  • jquery(function(){실행할내용;});
  • $(function(){실행할내용;})(jquery);
