제이쿼리 문서객체조작(DOM)_클래스조작

사용자가 선택한 요소에 클래스를 추가하거나 삭제, 또는 클래스가 없는 요소를 찾아서 서식을 적용하는 메소드

  • addClass(); - 선택한요소에 클래스서식을 추가
  • removeClass(); - 선택한요소에 클래스서식을 제거
  • toggleClass(); - 선택한요소에 클래스서식을 추가 / 제거
  • hasClass(); - 선택한요소에 해당 클래스이름이 있는지 여부 판단해서 true, fals 값으로 표시
  • Class(); - 사용자가 선택한 요소에 서식을 적용

버튼 클릭 시 box 요소에 서식을 추가 / 제거 하기
