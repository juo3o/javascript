이벤트(event)

EVENT : 특별한 상황, 행사, 사건

웹페이지에서 click, hover, out, drag, doubleclick, scroll, keybord 등의 행위

1. 이벤트 발생 종류

  • onclick(); - 마우스로 클릭
  • ondblclick(); - 마우스로 더블클릭
  • onkeydown(); - 키보드를 눌렀을 때
  • onkeypress(); - 키보드를 눌렀다 뗐을 때
  • onkeyup(); - 키보드를 놓았을 때
  • onmousedown(); - 마우스로 버튼을 눌렀을 때
  • onmouseup(): - 마우스로 버튼을 놓았을 때
  • onmouseover(); - 마우스로 버튼을 올렸을 때
  • onmouseout(); - 마우스로 버튼을 올렸다가 떼었을 때

모달윈도우 == 팝업레이어 == 팝업창

모달윈도우(modal window)는 사용자 인터페이스(UI)디자인 개념에서 자식윈도우에서 부모윈도우로 돌아가기전에 사용자의 상호동작을 요구하는 창.

모달윈도우는 부모창을 어둡게하고, 자식요소가 화면의 정가운데 배치. 오른쪽 상단에 '닫기'버튼을 만들어 닫을 수 있게 한다.
