함수(function)

함수(function) :
함수는 일련의 내용을 삽입하여 사용자가 언제든 호출하여 내용을 실행할 수 있는 실행문.
코드의 중복을 피할 수 있다. 재사용 용이. 유지보수에도 용이.
사용자가 호출하기 전까지 안나옴!!
함수의 종류에는 내장함수(alert, console.log, prompt...), 사용자정의함수(임의로 지은 함수명)가 있다.
        //함수문법 3가지 : 

        1. 매개변수 없는 일반 사용자정의함수
        function 함수명(){
            실행할내용;
        }

        2. 매개변수 있는 함수
        function 함수명(변수1, 변수2, 변수3, 변수4....){
            실행할내용;
        }

        3. 리턴값이 있는 함수
        function 함수명(변수1, 변수2, 변수3, 변수4....){
            실행할내용;
            return 실행결과;
        }
