자바스크립트 소개와 기초문법

  • 자바스크립트는 html 문서안에서 동작되는 언어이다.
  • 자바스크립트는 모든 명령어가 객체언어이다. (html - 태그/속성, css - 선택자/속성 , js - 객체/속성/메서드 ) 
    •                         객체(스마트폰)   - 메소드 (켜다/끄다)
                                         - 속성   (4.7인치/흰색)
  • html 문서의 head, body 영역에 작성한다.
  • script로 시작하여 /script로 닫아준다.
  • 자바스크립트는 주석을 작성할 때 2가지 종류가 있다. //한줄주석 , /*두줄이상주석*/
  • html, css는 대소문자를 구별 안하지만, 자바스크립트는 대소문자를 구별한다.
  • 자바스크립트에서 문자 데이터를 입력할 때는 ''(홑따옴표,싱글쿼테이션), ""(쌍따옴표,더블쿼테이션) 기호를 사용한다.
  • 자바스크립트는 css언어와 같이 내부스크립트, 외부스크립트가 존재한다.
  • 자바스크립트의 문서확장자는 '파일명.js'로 끝난다.
