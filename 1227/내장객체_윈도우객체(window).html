내장객체_윈도우(브라우저)객체(window)

브라우저객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용.

window 객체

window객체는 브라우저객체중 최상위객체로서 현재 실행되고있는 브라우저창을 말함.

window객체의 속성

  • document
  • frameElement
  • innerHeight
  • innerWidth
  • location
  • name
  • screenX
  • screenY
  • screenX, screenY
  • sessionStorge

window객체의 메서드

  • alert() - 윈도우창에서 경고창
  • blur() - 현재창에서 포커스 제거
  • close() - 현재창을 닫기
  • confirm() - 확인,취소버튼이 있는 확인창을 표시함
  • focus() - 현재창에서 포커스(초점) 부여
  • moveBy() - 현재창을 지정크기만큼 이동
  • moveTo() - 현재창을 지정좌표로 이동
  • open() - 새로운 창 열기
  • prompt() - 사용자가 입력가능한 프롬프트창 열기
  • resizeTo() - 동적으로 브라우저창 크기 조절
  • scroll() - 문서의 특정위치로 스크롤
  • sizeToContent() - 내용에 맞게 창의 크기를 조절
  • stop() - 로딩 중지

모달윈도(modal window)

모달윈도는 사용자 인터페이스(ui)디자인개념에서 자식 윈도에서 부모윈도로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다.

모달윈도는 부모창을 배경으로 어둡게하고 자식요소가 화면의 가운데 배치하도록 하며, 오른쪽상단에 '닫기'버튼을 만들어서 닫을 수 있도록 한다.
