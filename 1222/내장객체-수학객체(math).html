내장객체_ 수학(math)객체

수학객체는 다른 내장객체와 다르게 new 키워드를 사용하지 않는 '정적인 개체'

객체는(object)는 메서드, 속성으로 나눠진다.

메서드(동사) : 기능적인 동작. write();

속성(명사) : 서식. 정해진 기능. .style.color='서식내용'

수학객체의 속성들

  • e : 상수를 구함
  • PI : 원주율
  • SQRT2
  • SQRT1_2
  • LN2
  • LN10
  • LOG2E
  • LOG10E

수학객체의 메서드 종류

  • max(숫자1, 숫자2, 숫자3...) : 숫자중에서 최대값을 반환
  • min(숫자1, 숫자2, 숫자3...) : 숫자중에서 최소값을 반환
  • round(숫자) : 소수점 첫째 자리를 반올림
  • ceil(숫자) : 소수점 자리를 무조건 올림
  • floor(숫자) : 소수점 자리를 무조건 절삭
  • abs(숫자) : 수의 절대값을 반환
  • random(숫자) : 0과 1사이의 난수를 표시
