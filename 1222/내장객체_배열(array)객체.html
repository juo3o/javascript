내장객체_배열(array)객체

배열(array)객체 :
연관된 데이터를 하나처럼 관리하고 싶을 때 사용
복수의 데이터를 제어할 수 있는 다양한 함수를 제공
배열함수를 이용한 데이터의 편리한 조작

1. 배열객체 선언하기

방법1) let array = new Array(data1, data2, data3, data4 ... dataN);

방법2) let array = [data1, data2, data3, data4 ... dataN];

예 1) let fruits = new Array('수박', '참외', '귤' ...)

예 2) let fruits = ['수박', '참외', '귤' ....]

2. 배열객체 호출하기

인덱스 번호를 사용함. 번호는 0부터 시작한다

실습예제 ) 중화요리 이미지를 사용하여 요일별 음식 추천하기

날짜객체를 사용하여 현재 요일 번호를 구해서 배열번호에 맞는 이미지 파일 대입
