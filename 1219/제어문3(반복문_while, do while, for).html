제어문3(반복문_while, do while, for)

반복수행할 때 사용. 조건문(if)이 프로그램을 지적으로 만들고 반복문은 프로그램을 강력하게 만듬.

주의점 : 조건을 잘못 설정하면 무한루프에 빠지는 오류. 브라우저에 화면 멈춤현상이 발생.

  • while문 - 반복조건
  • do while문 - 반복조건
  • for문 - 반복횟수
        1. while문

                while(조건식){
                    실행할내용;
                }


        2. do while문

                do{
                    실행할내용
                }
                while(조건식)


        3. for문

                for(초기값;조건식;증감식){
                    실행할내용;
                }

실습 - 반복문을 사용하여 숫자 1부터 10까지 출력하기 (ex. 페이지번호, 글번호, 달력)

실습 - 반복문으로 구구단만들기 예) 2*1=2 , 2*2=4

실습 - 반복문으로 테이블태그 만들기 ( 달력 )
