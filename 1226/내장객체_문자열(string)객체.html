내장객체_문자열(string)객체

js에서 가장 활용도가 높은 객체. 문자열값을 인덱스번호를 통해 값 찾기나 다른 문자로 변경 가능하다

ex ) 아이디패스워드 입력창 문자열개수 체크, 단문메세지 문자열개수 체크, 패스워드 찾아서 다른 문자열로 변경, 이미지 갤러리 등..

문자열 객체는 다른 내장객체와는 다르게 new 키워드를 생략해도 된다.

문자열 객체의 메서드와 속성들

  • replace - 문자 치환하기
  • split - 문자열 분리하기
  • substring - 인덱스번호(0부터시작)부터 문자열 표시
  • length - 문자열 개수를 체크
  • indexof - 문자열과 일치하는 인데스번호를 반환
