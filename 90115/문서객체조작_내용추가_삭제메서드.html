문서객체조작_내용추가)삭제메서드

DOM(문서객체모델)에서 문서 안의 요소에 내용을 추가하거나 삭제 가능

  • append(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨뒤쪽에 내용추가
  • prepend(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨앞쪽에 내용추가
  • after(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨뒤에 내용추가
  • before(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨앞에 내용추가
ex. 실시간순위롤링. 뉴스속보롤링. 배너클릭시 이미지페이지 위로 나타남

insertAfter(), insertBefore()를 사용하면 A앞에 B를 배치할 수 있고, B앞에 A를 배치할수도 있다
after(), before()메소드와 기능은 같지만 사용법이 다르다

예)
$('A').after('B'); - A요소 바깥뒤에 B내용 추가
$('B').insertAfter('A'); - A요소 바깥뒤에 B내용 추가

문서객체조작 - 내용삭제 메서드(remove)

사용자가 선택한 요소의 내용만 제거하거나, 아예 태그요소를 삭제할 수 도 있다

  • remove(); - 사용자가 선택한 요소 제거
  • empty(); - 사용자가 선택한 요소의 내용만 제거
아래 remove버튼을 클릭하면 이 박스요소가 제거됩니다
아래 empty버튼을 클릭하면 이 박스요소 안의 내용만 제거됩니다
